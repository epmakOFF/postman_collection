# postman_collection
To start work with Yandex 360 API collection:
1. Import collection into your Postman: File -> Import -> Select API 360 Business.postman_collection.json
2. Create environment with the following variables:
   - OAuth-token
   - orgId
   - userId
   - service_app_id
   - service_app_secret
   - domain
   - record_id
   - groupId
   - type
   - id
   - ruleId
   - departmentId
   - alias
